Description

Tests erlang/mmerge.erl. Tests multi-way merge of k sorted lists of integers, total length n, in memory, using either an algorithm with runtime linear in k that is simple in the imperative language samples but not so in the current Erlang sample; or instead an algorithm that uses a priority queue with runtime logarithmic in k. Either way, the dependence on n is linear. Minimal error handling.
